## üß† Algoritmos de Machine Learning

### **1. Eigenfaces (An√°lisis de Componentes Principales)**

#### üìñ **Descripci√≥n T√©cnica:**
Eigenfaces es una t√©cnica de reducci√≥n de dimensionalidad basada en PCA (Principal Component Analysis) que representa rostros como combinaciones lineales de "caras propias" (eigenfaces).

#### üî¨ **Implementaci√≥n:**

#### ‚öôÔ∏è **Proceso Algor√≠tmico:**
1. **Preprocesamiento:** Normalizaci√≥n a 100x100 p√≠xeles, escala de grises, ecualizaci√≥n
2. **Centrado:** Sustracci√≥n de la cara promedio (mean face)
3. **PCA:** Descomposici√≥n en componentes principales
4. **Proyecci√≥n:** Representaci√≥n en espacio de eigenfaces reducido
5. **Clasificaci√≥n:** Distancia euclidiana en espacio reducido

#### üìä **Ventajas:**
- ‚úÖ Eficiente computacionalmente
- ‚úÖ Robusto ante variaciones de iluminaci√≥n
- ‚úÖ Compresi√≥n efectiva de datos

#### ‚ö†Ô∏è **Limitaciones:**
- Sensible a cambios de pose
- Requiere alineaci√≥n facial

### **2. Local Binary Patterns (LBP)**

#### üìñ **Descripci√≥n T√©cnica:**
LBP es un operador de textura que describe patrones locales en im√°genes mediante la comparaci√≥n de cada p√≠xel con sus vecinos, generando c√≥digos binarios invariantes a cambios de iluminaci√≥n.

#### üî¨ **Implementaci√≥n:**
```python

        self.radius = radius
        self.n_points = n_points
        self.grid_size = grid_size
```

#### ‚öôÔ∏è **Proceso Algor√≠tmico:**
1. **Patr√≥n Circular:** Radio=2, 16 puntos de muestreo
2. **Codificaci√≥n Binaria:** Comparaci√≥n con p√≠xel central
3. **Histograma Uniforme:** Solo patrones uniformes (‚â§2 transiciones)
4. **Divisi√≥n en Grilla:** 8x8 regiones para descripci√≥n espacial
5. **Concatenaci√≥n:** Vector final de 1152 caracter√≠sticas

#### üìä **Ventajas:**
- ‚úÖ Invariante a cambios de iluminaci√≥n monot√≥nicos
- ‚úÖ Robusto ante ruido
- ‚úÖ Describe caracter√≠sticas locales

#### ‚ö†Ô∏è **Limitaciones:**
- Sensible a rotaciones
- Mayor dimensionalidad

---
